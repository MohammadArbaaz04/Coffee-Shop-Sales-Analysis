# ☕ Coffee Shop Sales Analysis

## 📌 Project Overview
The objective of this project is to perform a detailed analysis of a Coffee Shop’s sales performance using **Advanced Excel**. This project identifies Key Performance Indicators (KPIs) such as revenue trends, peak consumption hours, and product popularity across multiple store locations.

The goal is to provide data-driven insights to optimize staffing during peak hours and manage inventory effectively by analyzing over **149,000+ rows** of transaction data.

---

## 🛠️ Tech Stack & Skills
* **Power Query:** Data Cleaning, ETL processes, and Date/Time formatting for 149K+ records.
* **Pivot Tables & Dashboards:** Multi-dimensional analysis to track sales by day, hour, and location.
* **Data Visualization:** Using Trend Lines, Bar Charts, and Heatmaps to identify peak sales activity.
* **Business Metrics:** Calculating Total Revenue, Average Order Value (AOV), and Category Contribution.

## 📈 Key Insights (Business Impact)
Based on the analysis of the transaction data from January to June, here are the primary findings:

| Metric | Insight |
| :--- | :--- |
| **Total Revenue** | **$698,812** generated from 149,116 transactions. |
| **Top Category** | **Coffee** is the dominant category, contributing **$269,952** in total sales. |
| **Peak Performance** | **Monday and Friday** are the busiest days, showing the highest footfall. |
| **Location Strategy** | **Hell's Kitchen** and **Astoria** are the top-performing store locations. |
| **Product Leaders** | **Brewed Chai Tea** and **Gourmet Brewed Coffee** are the highest selling items. |
| **Time Analysis** | Sales peak significantly between **8:00 AM and 10:00 AM**, indicating breakfast rush. |
